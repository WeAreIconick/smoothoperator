== Description ==

Smooth Operator is a WordPress Gutenberg block that provides automatic page scrolling with smooth animations and visual effects. Sit back and let your content do the scrolling - perfect for demos, showcases, and making your site look like it has a mind of its own.

Perfect for:
* Creating TikTok videos with smooth scrolling
* Recording screen captures with controlled scrolling
* Creating automated presentations and demos
* Showcasing content with engaging visual effects
* Any use case requiring smooth, controlled page scrolling

Features:
* Configurable scroll speed (pixels per second)
* Smooth continuous scrolling
* Play/pause controls on the frontend
* Automatically stops when reaching the bottom of the page
* Fixed position control button for easy access
* Responsive design for mobile devices
* Visual effects: parallax, particles, gradient shifts, blur transitions, scale effects, and glitch effects
* Content animations with multiple styles
* Typing effect for headings

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/autoscroll-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Run `npm install` in the plugin directory to install dependencies.
4. Run `npm run build` to build the block assets.
5. Use the block by adding it to any post or page through the block inserter.

== Frequently Asked Questions ==

= How do I use the block? =

1. Add the Smooth Operator block to your post or page
2. Configure the scroll speed and visual effects in the block settings panel
3. Publish or preview your page
4. Click the play button on the frontend to start scrolling
5. Click pause to stop scrolling

= What are good settings for TikTok videos? =

For smooth TikTok videos, try:
* Scroll Speed: 50-100 pixels per second
* Enable visual effects for more engaging content

Adjust based on your content length and desired video duration.

= Does it work on mobile? =

Yes, the block is fully responsive and works on mobile devices.
